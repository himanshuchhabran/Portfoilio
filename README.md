# Portfolio scaffold

Files created:

- `index.html` — main site
- `css/styles.css` — styling
- `js/main.js` — job title animation
- `assets/resume.pdf` — placeholder resume (replace with real PDF)

To preview locally, open `index.html` in your browser or run a simple HTTP server:

```bash
# Python 3
python -m http.server 8000

# then open http://localhost:8000
```

Deploy to GitHub Pages
---------------------

1. Create a new repository on GitHub and push this folder.
2. In the repository settings enable GitHub Pages from the `main` branch (root).
3. Your site will be available at `https://<username>.github.io/<repo>/` within a minute.

Create a PDF of the resume
--------------------------

1. Open `resume.html` in your browser.
2. Use the browser Print dialog and choose "Save as PDF" or "Microsoft Print to PDF".
3. Save the file as `resume.pdf` and place it in the `assets/` folder to enable the "Download PDF" button.
